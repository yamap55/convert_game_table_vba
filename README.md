# convert_game_table_vba
ExcelのVBA（Visual Basic for Applications）を使用し、特定のシート内容を元に新たに別のシートを作成する。
具体的には下記の勝敗表シートから出力シートを作成する。

## データサンプル
### original（勝敗表）
| No. | 氏名1 | 勝敗1 | 勝敗2 | 氏名2 |
|-----|-------|-------|-------|-------|
| 1   | A     | 〇    | ×     | B     |
| 2   | B     | ×     | 〇    | D     |
| 3   | C     | 〇    | ×     | A     |
| 4   | D     | ×     | 〇    | C     |
| 5   | B     | 〇    | ×     | A     |
| 6   |       |       |       |       |
| 7   |       |       |       |       |

### 結果（出力）
|   | A    | B    | C  | D |
|---|------|------|----|---|
| A | *    | 〇,× |    | × |
| B | ×,〇 | *    | ×  |   |
| D |      | 〇   | *  | × |
| C | 〇   |      | 〇 | * |
