# convert_game_table_vba
ExcelのVBA（Visual Basic for Applications）を使用し、特定のシート内容を元に新たに別のシートを作成する。
具体的には下記の勝敗表シート、名簿シートから出力シートを作成する。

勝敗表シートはそのまま使用するのではなく、以下の条件でデータを選別後に出力する。

## 選別条件
- 名簿に存在しない人との勝敗は使用しない
- 試合数が20に満たない場合は対戦表から削除する
